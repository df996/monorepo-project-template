## VUE3 项目模板

> 本项目强制使用pnpm<br />
> 使用husky+commitizen+lint-staged强制源码风格及git commit消息模板<br />

##### 安装：
```bash
# 如果没有安装pnpm需先安装
# npm i -g pnpm
pnpm install
```

___

##### 运行：
```bash
pnpm dev
```

##### 编译：
```bash
pnpm build
```

##### 核心依赖：
- vue3
- vite
- vue-router
- pinia

___

##### 默认使用的第三方库：
- element-plus
- axios
- dayjs
- tailwindcss
- normalize.css
- postcss
